## 隐私政策 (Privacy Policy)

**最后更新日期：2026年6月29日**

本《隐私政策》旨在帮助您了解：当您访问 summonhim.top 并使用本站提供的各项服务时，我们会收集哪些信息、如何使用这些信息，以及我们会采取哪些措施保护您的隐私。

当您注册或使用本站服务时，即视为您已阅读并同意本隐私政策。

### 1. 我们收集哪些信息

为了维持服务的正常运行与安全，我们可能会收集以下类型的信息：

- **账户信息：** 当您在本站注册时，我们会收集您的用户名、电子邮箱地址，以及经过单向加密（Hash）处理的密码。
- **连接与日志数据：** 当您访问本站各类服务（如代理、网盘、游戏服务器等）时，服务器会自动记录必要的系统日志。这些日志可能包括您的 IP 地址、浏览器类型（User-Agent）、访问时间、连接时长及设备信息。
- **用户生成内容：** 包括您主动存储、上传或发送的数据，例如文件、邮件、动态内容、播放记录等。

### 2. 核心服务的隐私边界说明（重要）

由于本站提供多种不同类型的服务，不同数据的可见性和控制边界并不完全相同，请您特别留意以下说明：

- **密码管理服务 - 零知识机制：**
  密码管理器采用端到端加密技术。您的密码、笔记等内容会先在您的本地设备上使用主密码加密，再上传至服务器。**服务器管理员无法解密、查看或恢复您的密码库内容。**
- **加密聊天服务 - 端到端加密：**
  如果您在聊天服务中启用了端到端加密，服务器仅负责转发加密后的数据，管理员无法读取聊天内容。
- **社交与邮件服务 - 对外传输风险：**
  当您向站外用户发帖、互动或发送邮件时，相关数据可能会被传输到其他外部服务器。**这些外部服务器不受本站控制，相关公开内容或跨服务器传输的数据不在本站可完全控制的隐私保护范围内。**
- **文件存储服务 - 管理权限边界：**
  您上传的文件通常以明文形式，或以系统级磁盘加密的方式存储在服务器中。**原则上，管理员不会主动查看或审查您的私人文件。** 但从技术权限上看，具备最高权限的管理员在特定情况下可能有能力访问这些数据，除非您自行进行了额外加密。
- **代理与网络服务 - 元数据记录：**
  为防止滥用及排查故障，我们会记录必要的网络连接元数据，例如某个 IP 的连接时间。**我们不会记录您的深度浏览内容，也不会长期保存具体查询明细；仅在短期故障排查场景下，才可能进行必要的技术抓包分析。**

### 3. 我们如何使用您的信息

我们承诺，收集的信息仅用于以下目的，**不会出售给任何第三方用于商业营销**：

1. 验证您的身份，并向您提供本站各项服务；
2. 监控服务器运行状态，开展故障排查与性能优化；
3. 防止服务被滥用，例如发送垃圾邮件或将代理服务用于网络攻击；
4. 在遭遇黑客攻击或系统故障时，进行安全处置与数据恢复。

### 4. 信息披露与共享

除以下情形外，我们不会主动向外部第三方披露或共享您的个人信息：

1. **法律要求：** 如遇所在地国家或地区执法机构提出合法调查请求，且案件涉及严重违法犯罪行为（如儿童色情、恐怖主义等），管理员有权依法提供相关账号日志及必要数据。
2. **保护服务器安全：** 如您的账号涉及严重危害服务器或网络安全的行为，我们可能会使用相关数据进行追踪，并向有关服务提供方报告。

### 5. 数据存储与安全

- 我们已采取合理的基础安全措施保护您的数据，例如 HTTPS 加密传输、防暴力破解及数据库权限隔离。
- **免责提示：** 本站为非商业性质平台。我们会尽力保障数据安全，但无法提供等同于大型商业机构级别的绝对安全保证。如发生不可抗力或高强度入侵导致的数据泄露事件，我们会尽快通知您，但不承担由此产生的赔偿责任。

### 6. 用户权利与账号注销

- 您可以随时登录相关服务，自行删除您上传的数据。
- 如您希望彻底注销统一认证账号，并删除在本站保存的相关数据，请通过邮件联系管理员。我们将在技术可行的范围内协助清理；但系统备份中的部分日志可能需要一定时间才会被覆盖删除。
